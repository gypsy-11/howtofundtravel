Search Functionality Test

This page tests the global search functionality in the header.

How to Test:

  1. Look for the search input in the header (top right)
  2. Type a search term like "remote work" or "online business"
  3. Press Enter or click the search icon
  4. You should be redirected to the blog page with search results

Test Search Terms:

  • remote work - Should find remote work articles
  • online business - Should find online business articles
  • investing - Should find investment articles
  • mindset - Should find mindset articles
  • affiliate - Should find affiliate marketing articles

Expected Behavior:

  • ✅ Search input should be visible in header
  • ✅ Search icon should be clickable
  • ✅ Enter key should trigger search
  • ✅ Search should redirect to blog with results
  • ✅ Placeholder text should rotate every 3 seconds
  • ✅ Focus effects should work on search input
